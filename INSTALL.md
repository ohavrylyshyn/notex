# Інструкція з встановлення

Цей документ містить покрокову інструкцію з встановлення та запуску NoteX.
Опис проєкту та основні можливості доступні в [README.md](./README.md).

## Крок 1: Встановлення Node.js

Node.js - це програмне середовище, яке дозволяє запускати JavaScript програми на вашому комп'ютері. NoteX потребує Node.js для роботи.

**Завантажте та встановіть Node.js:**

**Посилання:** https://nodejs.org/

**Рекомендується завантажити версію "LTS" (Long Term Support) - це найстабільніша версія.**

### Установка на Windows

1. Перейдіть за посиланням https://nodejs.org/
2. Натисніть велику зелену кнопку "Download LTS"
3. Буде завантажено файл `node-vX.X.X-x64.msi` (займе близько 50-100 МБ)
4. **Двічі натисніть** на завантажений файл `.msi`
5. Натисніть **"Next"** кілька разів, приймаючи всі параметри за замовчуванням
6. Натисніть **"Install"** та очікуйте завершення встановлення (2-3 хвилини)
7. Натисніть **"Finish"**

**Перевірка встановлення:**

- Натисніть **Windows + R**
- Введіть `cmd` та натисніть Enter
- У чорному вікні, що з'явилось, введіть:
    ```
    node --version
    ```
- Якщо все встановлено правильно, ви побачите номер версії (наприклад: `v20.10.0`)

### Установка на macOS

1. Перейдіть за посиланням https://nodejs.org/
2. Натисніть **"Download LTS"**
3. Вибіріть **"macOS Installer"** (`.pkg` файл)
4. Двічі натисніть на завантажений файл `.pkg`
5. Дотримуйтесь інструкцій установника
6. Введіть пароль вашого Mac, коли буде запропоновано
7. Натисніть **"Install"** та очікуйте завершення

**Перевірка встановлення:**

- Відкрийте **Terminal** (знайдіть через Spotlight: Cmd + Space, введіть "Terminal")
- Введіть команду:
    ```
    node --version
    ```
- Ви повинні побачити номер версії

### Установка на Linux

Для Linux користувачів найпростіший спосіб - використовувати менеджер пакетів:

**Ubuntu / Debian:**

```bash
sudo apt update
sudo apt install nodejs npm
```

**Fedora / CentOS / RHEL:**

```bash
sudo dnf install nodejs npm
```

**Arch Linux:**

```bash
sudo pacman -S nodejs npm
```

**Перевірка встановлення:**

```bash
node --version
npm --version
```

## Крок 2: Встановлення залежностей NoteX

Після встановлення Node.js вам потрібно встановити залежності програми NoteX.

### На Windows

1. Відкрийте **Провідник файлів** (Windows + E)
2. Знайдіть папку `notex` (там, де ви розпакували NoteX)
3. Натисніть в адресному рядку вверху (де показана директорія)
4. Виділіть весь текст та введіть: `cmd`
5. Натисніть **Enter** - відкриється командний рядок
6. Введіть команду:
    ```
    npm install
    ```
7. Очікуйте завершення (це займе 1-3 хвилини)
8. Ви повинні побачити повідомлення про успішне завершення

**Або альтернативний спосіб:**

1. Натисніть **Windows + R**
2. Введіть `cmd` та натисніть Enter
3. Перейдіть до папки з NoteX (наприклад):
    ```
    cd C:\Users\YourUsername\Downloads\notex
    ```
4. Введіть:
    ```
    npm install
    ```

### На macOS

1. Відкрийте **Terminal** (Cmd + Space, введіть "Terminal")
2. Введіть команду для переходу до папки NoteX:
    ```
    cd /path/to/notex
    ```
    (замініть `/path/to/notex` на реальний шлях до папки)
3. Введіть:
    ```
    npm install
    ```
4. Очікуйте завершення встановлення

**Легший спосіб:**

- Відкрийте папку NoteX у Finder
- Натисніть **Cmd + Shift + .** (щоб показати приховані файли)
- Найдіть файл `.zshrc` або `.bash_profile`
- Скопіюйте повний шлях до папки NoteX
- В Terminal введіть: `cd ` та вставте скопійований шлях

### На Linux

1. Відкрийте **Terminal** (зазвичай Ctrl + Alt + T)
2. Перейдіть до папки з NoteX:
    ```
    cd ~/Downloads/notex
    ```
    (або туди, де у вас розташована папка)
3. Введіть:
    ```
    npm install
    ```
4. Очікуйте завершення встановлення

## Крок 3: Запуск NoteX

Після встановлення залежностей запустіть програму:

### На всіх платформах:

У тому ж командному рядку / терміналі введіть:

```
npm start
```

NoteX з'явиться як жовте вікно на вашому робочому столі!

## Запуск

Для звичайного використання:

```bash
npm start
```

Для режиму розробки:

```bash
npm run dev
```

## Вирішення проблем

### "npm: команда не знайдена" або "npm: не розпізнана"

**Причина:** Node.js не встановлено або не додано до PATH.

**Вирішення:**

1. Переконайтеся, що ви встановили Node.js з https://nodejs.org/
2. **Перезавантажте комп'ютер** (важливо!)
3. Відкрийте командний рядок / Terminal заново
4. Спробуйте ввести `npm --version` знову

### "node_modules" папка не створена або помилка під час npm install

**Причина:** Можлива проблема з інтернет-з'єднанням або дозволами.

**Вирішення:**

1. Переконайтеся, що у вас активне інтернет-з'єднання
2. Видаліть папку `node_modules` (якщо вона існує)
3. Видаліть файл `package-lock.json` (якщо він існує)
4. Спробуйте ввести `npm install` ще раз

### Програма не запускається

**Вирішення:**

1. Переконайтеся, що ви ввели команду `npm start` в папці NoteX
2. Перевіріть, що папка `node_modules` існує (створена після `npm install`)
3. Закрийте командний рядок / Terminal і спробуйте заново

## Як знайти тип свого комп'ютера?

**Windows:**

- Натисніть Windows + Pause, ви побачите тип операційної системи

**macOS:**

- Натисніть логотип Apple в верхньому лівому куті - About This Mac

**Linux:**

- Введіть в Terminal: `uname -m`
